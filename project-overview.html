Introduction

Below is a list of some of the projects I have worked on during my time at Iowa State. It is by no means an exhaustive list. Instead it highlights projects involving teamwork or interesting technologies.

Current Projects

Senior Design - Iowa State ISERink

The ISU ISERink entails creating a virtual copy of the Iowa State University network for the purposes of training the internal IT department on responding to cyber security threats of all kinds. Besides creating the virtual environment in which this training will occur, our team is also working on a series of scenarios that the IT department can utilize.
The technologies involved in this project include ESXI and vSphere for the virtualization aspect and a yet to be determined web framework which we will use to create a portal for the scenarios.

Past Projects

PLANtastic (2015)

For my ComS 309 class, my group decided to make an extensible web application using Django and Python with Bootstrap and Jinja templates. This web application helps users plan trips, parties, and meetings by providing a suite of widgets that do everything from helping you find a ride to helping you split the bill.
We actually took our app for a test drive when we went on a group trip to Adventureland in Des Moines. This was one of the top projects of our class for the semester, and we nearly won the team spirit award as well. We really enjoyed working together and have discussed continuing this project at a later date.

Loanshark (2013)

This was one of the first major projects I was involved in at Iowa State University. For our SE 319 course, my team and I decided to make a desktop application in C# and XAML that would keep track of who the user had loaned books or media to. It featured changeable color schemes, a favorite items area, and more. It was also my first experience with git, which I then vowed to master. I now use git on an almost daily basis.
